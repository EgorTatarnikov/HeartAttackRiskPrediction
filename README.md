# Предсказание рисков сердечного приступа

В рамках проекта было проведено исследование медицинских данных пациентов, обучена модель машинного обучения для прогнозирования риска сердечного приступа, а также создан веб-сервис для прикладного использования модели.

Приложение реализовано с использованием **FastAPI** и предназначено для загрузки csv файлов с медицинскими параметрами, предсказания вероятности сердечного приступа и выгрузки результатов в формате csv или json.

---

**Веб-приложение** было доступно по адресу:  
http://www.heartattackriskprediction.ru/  
(сайт больше не поддерживается)

---

**`Инструкция по работе с сервисом предсказания рисков сердечного приступа.pdf`** демонстрирует работу приложения на примере.

**`Heart Attack Risk Prediction.ipynb`** — ноутбук с исследованием данных и обучением моделей.

**`tatarnikov_results.csv`** — предсказания, полученные моделью на предоставленной тестовой выборке.
